# ⚽ 世界杯竞猜池

朋友间小圈子世界杯淘汰赛竞猜，支持**比分池**和**胜负池**独立投注。⚠️ 仅供娱乐，所有金额为模拟数据，不涉及真实金钱交易。部署在 GitHub Pages，数据库使用 Supabase。

## 玩法规则

每场淘汰赛设有两个独立竞猜池，**每个池子单独计算赔率和结算，互不影响**。

| 池子 | 竞猜内容 | 中奖条件 |
|------|----------|----------|
| 🎯 **比分池** | 猜 90 分钟常规时间比分 | 比分完全一致 |
| 🏆 **胜负池** | 猜哪支队伍晋级 | 选中晋级队伍 |

### 投注规则

- 每注金额（模拟）：最低 ¥50，最高 ¥200
- 每人在每个池子中只能有一条投注（新投注会自动覆盖旧的）
- 截止时间：默认**开赛前 3 小时**，管理员可为单场比赛自定义封盘时间（如决赛改为开球后 5 分钟）；**一旦封盘，封盘时间由数据库触发器锁定，不可再修改**（防管理员作弊）
- 截止后自动公布赔率和所有投注明细（公开透明）
- 管理员可隐藏已结束/投注过的比赛，比赛列表只显示进行中的场次（「我的投注」中的历史记录不受影响）

### 赔率计算

- 赔率 = 该池投注总额 ÷ 该选项投注额
- 例如：比分池总投注 ¥1000，其中 "2:1" 投注 ¥200，则 "2:1" 赔率为 5.0
- 赔率仅供参考，所有资金收付**线下进行**，本平台仅做记录和统计

### 退款规则

| 情况 | 处理 |
|------|------|
| 该池无人猜对 | 全部退款 |
| 该池所有人都押同一个选项（无对手盘） | 全部退款 |
| 有人猜对且有对手盘 | 赢家按赔率获得收益（线下结算） |

## 使用流程

### 普通用户

1. 打开页面，输入**房间密码**进入
2. 首次使用需**注册投注身份**：填写姓名 + 投注密码 + 确认密码
3. 之后每次输入姓名和密码登录即可
4. 选择比赛，在比分池或胜负池中填写竞猜内容和金额，点击投注
5. 截止前可点击投注旁的 ✕ 删除修改
6. 在「我的投注」标签查看自己的投注记录和结算结果

### 管理员

1. 进入「管理」标签，输入管理员密码
2. **添加比赛**：填写对阵双方和比赛时间
3. **录入赛果**：比赛结束后填写 90 分钟比分 + 晋级队伍，点击录入后自动结算
4. **查看投注**：可按场次或全局查看所有人的投注记录
5. **隐藏比赛**：已结束的比赛可一键隐藏，用户列表不再显示
6. **修改封盘时间**：每场可单独设置封盘时间（默认开赛前3小时），封盘后自动锁定

## 自行部署

### 1. 创建 Supabase 项目

在 [supabase.com](https://supabase.com) 免费注册并创建项目。

### 2. 初始化数据库

进入 Supabase SQL Editor，运行 `init.sql` 中的全部内容。

### 3. 配置前端

修改 `index.html` 中的配置项：

```javascript
const SUPABASE_URL = 'https://你的项目ID.supabase.co';
const SUPABASE_KEY = '你的anon key';
```

修改 `init.sql` 中的密码哈希后重新运行，或直接替换 `room_settings` 表中的值。

### 4. 部署到 GitHub Pages

将 `index.html` 推送到 GitHub 仓库，在 Settings → Pages 中开启即可。
